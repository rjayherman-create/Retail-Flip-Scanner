Retailer-specific markdown logic:

Costco:
- .97 = clearance / markdown signal
- .00 = possible local manager markdown
- .88 = possible markdown signal
- .99 = normal retail
- star on tag = possible discontinued / limited stock

Walmart:
- yellow clearance tag = strong markdown signal
- price ending .00 or .50 may be clearance
- compare regular price vs clearance price
- stronger score when clearance is 40%+ off
- strongest when item is small, sealed, giftable, and has known resale demand

Target:
- clearance percent is important
- 30% off = weak/maybe
- 50% off = maybe/good
- 70% off = strong
- 90% off = very strong but may be picked over
- use DPCI/UPC/TCIN when visible
- watch for seasonal and toy reset clearance

Scoring logic:
- Start at 50
- Add points for strong markdown
- Add points for high-demand category
- Add points for sealed/new condition
- Add points for easy storage
- Add points for expected profit above default minimum
- Subtract points for bulky item
- Subtract points for damaged box
- Subtract points for niche item
- Subtract points for low profit spread
- Subtract points for likely local saturation

Default recommendations:
BUY:
- score 75+
- expected profit at least $7 per unit
- item is easy to store and sell locally

MAYBE:
- score 55 to 74
- buy only 1 test unit

SKIP:
- score below 55
- low spread, damaged, bulky, niche, or slow resale risk
